Security Log Analysis & Alerting Tool

2025 • ~7 min read

Modern Security Operations Centers (SOCs) rely heavily on log analysis to detect, investigate, and respond to security incidents. One of the most common real-world attack patterns faced by Linux servers is SSH brute-force authentication attempts.

To better understand how such attacks are detected in practice, I built a Python-based Security Log Analysis & Alerting Tool that processes real SSH authentication logs, identifies suspicious login behavior, and generates SOC-style alert reports.

Project Overview

The tool analyzes Linux SSH authentication logs to detect repeated failed login attempts, correlate activity by IP address and username, and assign severity levels based on defined thresholds.

Key Features

  • Parsing real SSH authentication logs
  • Detection of brute-force login attempts
  • Correlation by IP address and username
  • IPv4 and IPv6 support
  • SOC-style alert reports (text & CSV)
